Rhetorlist

Tracking new books in Rhetoric & Writing, Composition Studies, Technical Communication, and related disciplines.

FAQ

What is this?

Rhetorlist is my attempt to track new book releases (scholarly monographs and edited collections) in writing research. I've too often missed the announcement for a great new book, only to find it months (or years!) later. By building a website that lists new books, I hope that I can solve my problemâ€”and help others hear about new book-length research in the field. I hope you find it useful. You can read more about the project's initial design in Building Rhetorlist.

Why isn't [insert subject/subdiscipline/area-of-inquiry] included?

I have no interest in policing the boundaries of writing, rhetoric, and literacy-related research. If you think a book should be included, please use the new book form. I'm happy to add it.

Do I have to come back to this site to see updates?

If you use an RSS Feed Reader that supports JSON Feed (I recommend Feedbin; if you'd prefer a free option, try Net News Wire or Inoreader), you can subscribe to this site and get updates when a new book is posted! Just point your feed reader to this site's JSON Feed.

What is OA and why is it a filter option?

Open Access. I believe in celebrating and circulating open access work, so I've chosen to highlight it here. And if you want something to immediately read, you can sort the Availability filter for OA books.

Why aren't you including articles, book reviews, or other scholarly forms?

I think it's much easier to find new articles than to find newly published books. Also: Tracking other scholarly forms would greatly increase the complexity of this project, and I am but one person.

Can I use the data for my own research project?

Absolutely. I hope that, over time, this site can tell us a bit about how our field produces monographs. If there's anything I can do to help, please email me.

There's a lot of information here. How is it stored?

I'm storing the site's data in json files. If you'd like to view and tinker, visit the project's repository on Github.

Can I make a variation of this for my scholarly field?

Sure! Clone the Github repository and make it your own.

How did you make this?

I buiilt the intial version of Rhetorlist (in 2018) with a javascript table library. In 2025 I shifted to a cover-based design, which I made with a lot of help from Claude Code and my coffee maker.
